YO-DETR: Lightweight End-to-End Visual Measurement with Adaptive Multi-Scale Fusion for Industrial Fault Detection


This code is associated with the manuscript "Lightweight End-to-End Visual Measurement with Adaptive Multi-Scale Fusion for Industrial Fault Detection" submitted to The Visual Computer.

We construct a dedicated dataset for modular air-cooling equipment fault detection containing 3,600 images and 22,307 annotated instances. Dataset Address https://pan.baidu.com/s/1mLf3fR3Y0crgTlng2-I53Q?pwd=knth
